# MenuComprasCS

Simple console application written in C# to manage a small shopping list. The project was used as practice to improve logic, menu navigation, input handling and basic file persistence concepts.

## What It Does

- Adds items to a shopping list
- Lists all items
- Searches for items
- Removes items
- Saves and loads items using a CSV file

## Technical Concepts Used

- C# (.NET Console App)
- Basic OOP & structured logic
- CSV file read/write
- Console input/output
- Collections
- Conditionals & loops
